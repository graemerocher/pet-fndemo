## Oracle Function + Micronaut Demo

Push a native image:

```
docker login us-ashburn-1.ocir.io
./gradlew dockerPushNative
```

Push JIT version:

```
docker login us-ashburn-1.ocir.io
./gradlew dockerPush
```
